ONG Educação - Projeto de entrega (Fundamentos HTML5)
Arquivos incluídos:
- index.html (página inicial)
- projetos.html (lista de projetos e área de doações)
- cadastro.html (formulário complexo com validação e máscaras)
- css/style.css
- js/script.js
- assets/* (imagens em SVG)

Instruções:
1) Extraia o zip e abra index.html no navegador.
2) Para disponibilizar no GitHub Pages: crie um repositório público e envie todos os arquivos,
   em seguida ative GitHub Pages nas configurações do repositório (branch main / root).
3) Os inputs têm validação nativa e máscaras em JavaScript (arquivo js/script.js).
4) Para validar no W3C: acesse https://validator.w3.org/ e carregue os arquivos HTML.

Paleta de cores pensada para educação: azul (#1E88E5) com acento amarelo (#FFD54F) — alta legibilidade.

Observações:
- As imagens estão em SVG (formatos vetoriais otimizados). Você pode incluir versões adicionais em JPG/WEBP na pasta assets se quiser.
- Este projeto foi criado com foco em semântica, acessibilidade (aria-labels, roles, fieldset/legend) e responsive design mobile-first.
